This Template was taught in a course taken from Coursera.

-> Download Project
-> Open the project location in preffered CMD
-> Type "npm install" and hit enter to install the node modules
-> Open the "json_server" location in preffered CMD
-> Type "json-server --watch db.json -p 3001 -d 2000" and hit enter to launch the json server
-> Type "npm start" and hit enter to launcha the website
